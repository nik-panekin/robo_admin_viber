
Данный скрипт Python представляет собой приложение (бот) для автоматической
модерации сообщений пользователей в сообществах Viber. Программа удаляет все
сообщения, содержащие web-ссылки.

Основная особенность скрипта: официальные Viber API для разработки ботов здесь
не используются (равно как и любые другие API аналогичного назначения), т.к.
на момент написания приложения в них отсутствовал требуемый функционал для
работы с сообществами Viber. Применён подход на основе алгоритмов машинного
зрения и распознавания символов (OCR). Используются open-source библиотеки
OpenCV и Tesseract.

Главный исполняемый файл скрипта называется robo_admin_viber.py. Журнал ошибок
и сообщений находится в папке logs.

Системные требования к работе скрипта:
    - операционная система MS Windows 7 или новее;
    - установленный клиент Viber в _русскоязычной_ локализации;
    - интерпретатор Python 3.8.6 или новее;
    - набор пакетов языка Python, перечисленных в файле requirements.txt;
    - система оптического распознавания Tesseract OCR v5 + файлы .traineddata
      для английского (eng) и русского языков (rus).

Для установки библиотеки Tesseract следует скачать, а затем инсталлировать
подходящий дистрибутив:

    https://digi.bib.uni-mannheim.de/tesseract/

Прямые ссылки на последние на момент написания программы версии Tesseract:

    https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w32-setup-v5.0.0.20211201.exe
    https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-v5.0.0.20211201.exe

Языковые файлы находятся здесь:

    https://github.com/tesseract-ocr/tessdata

Достаточно скачать файл rus.traineddata и скопировать в папку tessdata
установочной директории Tesseract (английский язык доступен по умолчанию).

Очень важно прописать путь к установочной папке Tesseract в системную
переменную PATH, а также создать ещё одну переменную с именем TESSDATA_PREFIX,
в которую записать абсолютный путь к папке tessdata.

Запуск данной программы осуществляется в следующей последовательности:

    1. Запустить приложение Viber, авторизовавшись как администратор
       сообщества. Выбрать требуемое сообщество в меню слева, чтобы его чат
       отображался в центральной области окна Viber. Скрыть правую панель с
       информацией о сообществе, чтобы большую часть окна занимал только чат.

       Рекомендуется окно Viber расположить так, чтобы оно занимало 3/4
       пространства рабочего стола сверху. Остальную 1/4 часть снизу будет
       занимать терминал-консоль с запущенным скриптом Python.

    2. Открыть консоль, расположив её так, чтобы были видны одновременно окна
       консоли и приложения Viber. Запустить скрипт Python командой:

       robo_admin_viber.py

       Через некоторое время фокус ввода должен автоматически вернуться в
       главное окно Viber. Если этого не произошло (о чём будет выдано
       предупреждающее сообщение), то установить фокус вручную, сделав щелчок
       мышью по заголовку окна Viber.

Для управления работой приложения используются комбинации клавиш:

    Ctrl + Alt + F10 - завершить работу скрипта;
    Ctrl + Alt + F12 - приостановить/возобновить работу скрпита.

Данные "горячие" клавиши будут срабатывать даже если окно консоли не в фокусе.

Используйте и/или модифицируйте данный программный код без ограничений.
